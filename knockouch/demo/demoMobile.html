knockouch.js

Knockouch is a javascript library that adds touch bindings to knockoutjs library. So you can work with touch events in knockout way.

You can download the sources from github or use nuget or just type

Install-Package knockouch

in package manager console. Note, that nuget adds hammerjs as a default touch library.

for knockouch you need bind touch events with knockout bindings for example u can see how we use it for this demo
